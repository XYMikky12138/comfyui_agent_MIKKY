═══════════════════════════════════════════════════════════
  动态输入端口 - 3 步快速上手
═══════════════════════════════════════════════════════════

第 1 步: 修改数量
─────────────
找到 inputcount 参数，修改为你需要的图片数量。
例如: inputcount = 5


第 2 步: 点击按钮 ⭐⭐⭐
─────────────
在节点上找到 "Update inputs" 按钮，点击它！

(或者右键节点 → 选择 "Update inputs")


第 3 步: 连接图片
─────────────
现在你会看到对应数量的输入端口：
- images_0
- images_1  
- images_2
- images_3
- images_4

连接你的图片，开始使用！


═══════════════════════════════════════════════════════════
  重要提示
═══════════════════════════════════════════════════════════

⚠️  修改 inputcount 后，一定要点击 "Update inputs" 按钮！

⚠️  这是 ComfyUI 的标准机制，不会自动刷新！

⚠️  如果看不到按钮，重启 ComfyUI 并刷新浏览器！


═══════════════════════════════════════════════════════════
  测试步骤
═══════════════════════════════════════════════════════════

1. 重启 ComfyUI
2. 刷新浏览器 (Ctrl+Shift+R)
3. 添加 BizyAir 节点
4. 应该看到 "Update inputs" 按钮和 2 个端口
5. 修改 inputcount = 5
6. 点击 "Update inputs" 按钮
7. 端口变为 5 个
